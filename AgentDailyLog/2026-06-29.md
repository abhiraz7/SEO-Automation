### Template Data Contract (`project_detail.html`)
The template receives these variables from `projects.py → project_detail()`:
- `project` — Project ORM object (`.id`, `.name`, `.base_url`, `.project_type`, `.created_at`)
- `pages` — list of Page ORM objects (`.url`, `.updated_at`, `.issues`, `.status_code`, `.title`, etc.)
- `page_data` — list of dicts: `{page, issues, suggestions, sugg_by_issue, title_issues, checklist, title_len, meta_len}`
- `grouped_issues` — `{category_str: [{page, message, severity, issue_id}]}` sorted by error count desc
- `semrush_connected` — bool
- `semrush_data` — dict (empty if no API key)
- `last_crawled_ago` — human string e.g. "3h ago" or "Never"
